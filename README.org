- [[https://gnome-terminator.org/][Terminator]] terminal emulator.
- [[https://github.com/hlissner/doom-emacs][Doom]] emacs.
- VIM and Neovim.
- Bash and bash aliases.
- Krita configuration and shortcuts.
- KDE keyboard shortcuts.
